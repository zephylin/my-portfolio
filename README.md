# Zephylin Dusengimana — Portfolio

Personal portfolio website showcasing my work in software development, AI/ML engineering, and research.

## Pages

- **Home** — Introduction and quick links
- **About** — Education, experience, skills, and awards
- **Projects** — Curated portfolio of technical projects
- **Research** — Research experience, interests, and publications
- **Contact** — Contact form and information

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla — no frameworks)
- Google Fonts (Inter)
- EmailJS (contact form)
- Hosted on GitHub Pages

## Project Structure

```
Portfolio/
├── index.html          Home page
├── about.html          About / Resume
├── projects.html       Project portfolio
├── research.html       Research & publications
├── contact.html        Contact form
├── assets/
│   ├── css/style.css   All styling
│   ├── js/main.js      Shared nav, footer, theme toggle
│   ├── js/contact.js   Contact form logic (EmailJS)
│   ├── images/         Photos, logo
│   └── docs/           Resume PDF, CV PDF
├── sitemap.xml
├── robots.txt
└── README.md
```

## Local Development

Open `index.html` in a browser. No build step required.

## Deployment

Push to `main` branch → GitHub Pages serves it automatically.
